## 📌 Sobre o projeto

O **PortfolioHUB** é o resultado do desafio *Implantação PortfolioHUB + IA Gemini*.
É um site estático que centraliza minha presença profissional e **lista meus repositórios
automaticamente, consumindo a API pública do GitHub** — novos projetos aparecem no portfólio
sem necessidade de edição manual.

A implantação foi guiada com apoio do **Google Gemini** e documentada em
[`docs/IMPLANTACAO.md`](docs/IMPLANTACAO.md).

## ✨ Funcionalidades

- 🏠 **Página inicial** com apresentação, destaques (robótica/olimpíadas) e habilidades.
- 📂 **Projetos dinâmicos** carregados em tempo real do GitHub (`/users/Arthur259493/repos`).
- 📄 **Currículo** com formação, experiência e competências.
- ✉️ **Contato** com links diretos e formulário via `mailto`.
- 🚀 **Deploy automático** no GitHub Pages via GitHub Actions.
